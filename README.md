## &nbsp;📌&nbsp; Featured Work

> A portfolio of quant-finance &amp; ML systems written from scratch — each with a clean public API, a real test suite, green CI, and a math write-up. ⭐ = has a visual walkthrough in its README.

### &nbsp;📈&nbsp; Quantitative Finance

| Project | What it is |
| :-- | :-- |
| ⭐ **[lobster](https://github.com/superkush06/lobster)** | Limit order book microstructure simulator — price-time-priority matching, latency & market-impact models, pluggable agents, **and replay of real NASDAQ LOBSTER message data** |
| ⭐ **[vol-surface](https://github.com/superkush06/vol-surface)** | Implied volatility surface — Black-Scholes, SABR (Hagan) + **multi-expiry SVI calibration with calendar no-arbitrage**, 3-D surface plots |
| ⭐ **[factor-zoo](https://github.com/superkush06/factor-zoo)** | Equity factor reproductions — momentum / value / quality / low-vol, **Fama-MacBeth regressions**, information-coefficient tearsheets |
| **[portopt](https://github.com/superkush06/portopt)** | Portfolio optimization — Markowitz, Black-Litterman, risk parity, Ledoit-Wolf shrinkage |
| **[garch](https://github.com/superkush06/garch)** &nbsp;·&nbsp; **[risk](https://github.com/superkush06/risk)** &nbsp;·&nbsp; **[kalman](https://github.com/superkush06/kalman)** &nbsp;·&nbsp; **[optune](https://github.com/superkush06/optune)** | GARCH/EGARCH vol models · VaR / CVaR & stress testing · Kalman / EKF / UKF filters · options pricing + Greeks via AAD |

### &nbsp;🤖&nbsp; Machine Learning

| Project | What it is |
| :-- | :-- |
| ⭐ **[gauss-bandit](https://github.com/superkush06/gauss-bandit)** | Multi-armed **and contextual** bandits — Thompson, UCB1, EXP3, **LinUCB** — with rigorous regret analysis |
| **[transformer-from-scratch](https://github.com/superkush06/transformer-from-scratch)** | A GPT-style decoder in pure NumPy with **manual backprop** — gradients verified to machine precision |
| **[tinydiff](https://github.com/superkush06/tinydiff)** | A reverse-mode autodiff engine in < 500 lines — trains real MLPs, no PyTorch |
| **[rl-gym](https://github.com/superkush06/rl-gym)** &nbsp;·&nbsp; **[bayes](https://github.com/superkush06/bayes)** &nbsp;·&nbsp; **[regimes](https://github.com/superkush06/regimes)** | Q-learning / SARSA / REINFORCE / A2C · MCMC (HMC) + variational inference · HMM regime detection |
